# SISMOB

## OBJETIVO
O SISMOB é um sistema de controle imobiliário.
A ideia inicialmente era informatizar os processos do setor que cuida dos patrimônios da PortosRio, permitindo aos usuários registrar os imóveis e bens pertencentes a empresa.


## TECNOLOGIAS UTILIZADAS

+ Django e DRF
+ React
+ MySQL

## DESENVOLVIMENTO & APRENDIZADOS

O SISMOB foi o primeiro software que desenvolvi no qual o objetivo principal não era apenas a prática ou aprendizado de uma tecnologia, mas sim um uso real e uma substituição de um processo que não era antes informatizado.
O projeto foi a porta de entrada para o início dos meus estudos na área de desenvolvimento WEB e por já possuir uma bagagem em python, decidi escolher o Django como framework para a criação do site.
Inicialmente o desenvolvimento ocorreu de maneira flúida, porém, com o tempo, a manutenção da solução padrão do django para renderizar o frontend se tornou uma tarefa ineficiente.

Procurando uma alternativa, comecei a avaliar a possibilidade do uso de React para a criação do frontend e comprei um curso para começar meus estudos.
Acabei aprendendo bastante sobre Javascript no processo.
Num certo ponto percebi que o conhecimento que tinha adquerido até então já era o suficiente para transformar os templates django (html) em uma aplicação React.
No entanto, mais um elemento foi necessário para fazer a integração entre os dois frameworks, esse elemento foi o Django Rest Framework (DRF).

O DRF foi a última grande peça do quebra-cabeças. Tendo agora um desenvolvimento independente do frontend, backend e uma forma eficiente de integrar o dois, a manutenção se tornou bem mais simples.
